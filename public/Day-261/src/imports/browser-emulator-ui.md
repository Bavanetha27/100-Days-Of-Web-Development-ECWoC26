Design a modern web application UI titled “Browser-Based Emulator.” The interface should simulate running an operating system or virtual machine directly inside the browser. Users should be able to start, stop, and interact with the emulator, view system resources, and access a terminal or display screen. The design should feel similar to online VM platforms, OS simulators, or retro console emulators, with a clean, developer-focused layout.
