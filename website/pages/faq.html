Find answers to common questions about the 100 Days of Web Development project, contributions, certificates, and usage.

It is a learning challenge where participants build projects daily using HTML, CSS, and JavaScript to improve frontend skills.

You can fork the repository, create a branch, add your improvements, and submit a pull request following the CONTRIBUTING guidelines.

Anyone interested in learning web development can participate, regardless of experience level.

Contributors can generate certificates from the Certificate Generator page using their GitHub username.
